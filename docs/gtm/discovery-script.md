# ICP discovery script (12–15 calls)

**Audience:** Editors + designers (and the producers who feed them) who regularly pull media from gated sources—their own IG/FB/Drive, members-only boards, private client links—into local Premiere/Resolve/FCP projects.

## Opener

“We’re building Rippopotamus so you can save full-quality media from anything you’re logged into—your own IG, FB, Drive, gated client boards—straight into editor-ready folders, all local. Not another sketchy converter.”

## Core questions

1. Where does the stuff you need to download live—and how much of it is behind a login?
2. How do you grab gated/members-only/private-board assets today? What breaks (login walls, quality caps, wrong filenames)?
3. How long does intake take per project, and where do files end up?
4. What would “done” look like for handoff into the edit?

## Pass / fail

- **Pass:** Regularly hits login walls / gated sources; cares about full quality + local files + client confidentiality; pain on intake > 30 min/month.
- **Fail:** Only ever downloads fully public one-offs; happy with a browser extension; cloud-only workflow with no local edit.
